# Seeing Like a State

Scott (1998). Core argument: states simplify complex local practices into legible, standardized forms in order to monitor and control them. This process of "legibility" destroys local knowledge (metis) and replaces it with centralized, abstract representations.

Chapter 3 — legibility as precondition for control. The state must see before it can act. Standardized names, cadastral maps, rational forests — all are technologies of legibility.

Connection to proxy-target: legibility devices are proxies. The map is not the territory. The standard is not the practice. And when the standard is enforced, the practice reshapes to match the standard rather than the underlying need.
