# Legibility is a precondition for both control and care

Scott's "Seeing Like a State" argues that legibility — making a complex system readable from a central vantage point — is a precondition for state control. The state simplifies (standardizes names, straightens roads, rationalizes forests) in order to see, and it sees in order to control.

But legibility is also a precondition for care. You can't care for what you can't see. A doctor needs legible symptoms. A teacher needs legible student understanding. A parent needs legible emotional states. The same mechanisms that enable control also enable stewardship.

This connects to the Goodhart/proxy discussion: making something legible means creating a proxy for it. The metric, the dashboard, the report — these are legibility devices. And they are subject to the same proxy-target dynamics. The question is whether the system that uses them is oriented toward control (in which case Goodhart's Law bites hard) or toward care (in which case there may be self-correcting feedback).
